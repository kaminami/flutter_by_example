# connectivity_plus_example

connectivity_plusを用いてネットワーク接続状態(mobile | wifi | none)を取得するサンプルアプリ。
状態の同期取得、非同期取得、riverpod_generatorを用いた非同期取得の3例を含む。

## 使用ライブラリ

- connectivity_plus
  - https://pub.dev/packages/connectivity_plus
