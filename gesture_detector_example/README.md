# gesture_detector_example

GestureDetectorを利用した、ドラッグによるWidget任意配置の実装例


# 備考

- Draggableを用いる場合と同様のことができる。
- GestureDetectorを使用するとDrag以外の操作にも対応できる。基本的にはDraggableではなく、GestureDetectorを使用するとDrag以外の操作にも対応できるを使用するのがよいと思われる。
